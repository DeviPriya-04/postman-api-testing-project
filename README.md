# 🚀 Postman API Testing Project

## 📌 Overview
This project demonstrates API testing using Postman. It includes a Postman Collection and Environment file to test REST APIs efficiently.

---

## 🛠️ Tools Used
- Postman
- Git
- GitHub

---

## 📂 Project Files
- `My Collection.postman_collection.json` → API test requests
- `Dev Environment.postman_environment.json` → Environment variables

---

## 📡 API Testing Features
- GET requests for fetching data
- JSON response validation
- Environment-based testing
- Organized collection structure
